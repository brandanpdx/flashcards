JS Flashcards

Javascript:

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators:

performs some operation on single or multiple operands (data value) and produces a result. For example 1 + 2, where + sign is an operator and 1 is left operand and 2 is right operand.

Variables:

hold the data value and it can be changed anytime. JavaScript uses reserved keyword var to declare a variable.

Functions:

is a subprogram designed to perform a particular task. Functions are executed when they are called. This is known as invoking a function.

Methods:

actions that can be performed on objects. A JavaScript method is a property containing a function definition. Methods are functions stored as object properties.

Arguments:

a local variable accessible within all functions and contains an entry for each argument passed to that function.
